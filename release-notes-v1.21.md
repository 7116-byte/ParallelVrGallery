# v1.21 系统图库式首页与相册/生成页

- 首页改为 `全部 / 相册 / 生成` 三入口，`生成` 替代旧生成管理主入口。
- `全部` 按日期分组显示系统图片和视频。
- `相册` 按 MediaStore 系统相册/文件夹聚合，支持进入单个相册浏览。
- `生成` 只显示已生成、生成中、暂停或失败的图片/视频 VR 缓存任务。
- Viewer 左右滑动范围跟随来源：全部、当前相册或生成集合互不串台。
- 更新 README，恢复中文优先说明并列出第三方项目来源。

下载 APK 后覆盖安装即可。模型文件仍使用单独的模型资产 Release，本地已有模型时不会因为普通更新而重新下载。
