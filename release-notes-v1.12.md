# v1.12

这版按新的版本线发布，后续修复和功能更新都从 `v1.12` 往后顺着走。

- 图片自动预加载最大只扩到前后 8，不再因为补位一路排到几百张之后。
- 视频帧拆分缓存保存到 App 私有持久目录，App 被杀后已生成帧不会被系统 cache 清理。
- 视频生成增加独立 CPU/GPU 设置，不再复用图片模型设置。
- 视频进度行增加前三帧平均生成时间，例如 `帧生成 1234ms`，时间进度放最右。
- 视频加入队列后按钮会变成“已在队列 / 生成中 / 已生成 / 重试视频 VR”。
- 通知栏进度把百分比放到标题和子文本里，不用长按也更容易看到。
- 视频 worker 改为进程级任务，进入后台后会继续尽量生成视频；图片预加载仍可随 VR 状态停止。
- 修复视频双击快进“识别了但原地跳转”的问题，改用底层播放器 seek。
- 视频页底部合并为一条控制区：播放/暂停、生成状态、时间和进度条放在一起。
- 检查更新结果提供“下载更新”按钮，会直接打开 GitHub APK 下载链接或 Release 页面。

已验证：`assembleDebug` 构建通过。
