# v1.23 大图库卡顿与内部生成库修复

- 修复 v1.21 首页改造后，大图库打开会卡住的问题。
- 移除主界面未选择状态下的全量 `indexOfFirst` 扫描，避免几万张媒体触发近似 N² 主线程计算。
- 首次打开只从 MediaStore 读取最近 1800 项媒体，避免启动阶段等待几万条记录；生成页仍可独立显示内部缓存。
- 全部/相册时间线改为渐进渲染：首屏显示 1200 项，继续下滑自动加载更多。
- 生成页开始按 App 内部生成库思路恢复缓存项：即使原图/原视频不在本次系统相册扫描结果里，也能从缓存目录恢复为生成页专用虚拟项。
- 内部生成库虚拟项不会出现在普通 `全部` 和 `相册` 页面，避免污染图库。

这版建议大图库用户直接覆盖安装。
